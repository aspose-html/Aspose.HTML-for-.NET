# Aspose.HTML for .NET - Installation

Aspose.HTML for .NET can be installed in several ways depending on your development environment. The recommended method is via NuGet Package Manager, but manual installation is also supported.

## 1. Install via NuGet (recommended)

**Package Manager Console**
```c#
Install-Package Aspose.HTML
```

**.NET CLI**
```c#
dotnet add package Aspose.HTML
```

This automatically downloads the latest version and adds all required dependencies to your project.

## 2. Install via NuGet UI (Visual Studio)

- Search for **Aspose.HTML**
- Click **Install**

## 3. Manual Installation

You can [download the library as a ZIP archive](https://releases.aspose.com/html/net/):

- DLLs for .NET Standard / .NET Framework
- Example projects

After downloading, extract the archive and reference `Aspose.HTML.dll` manually in your project.

## 4. System Requirements

Aspose.HTML for .NET supports:

- .NET 6+
- .NET Standard 2.0+
- .NET Framework 4.6.1+
- Windows, Linux, and macOS

Additional details are available in the official docs.
