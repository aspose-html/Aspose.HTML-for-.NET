# How to Run the Examples?

The package contains C# examples. Follow the given steps to proceed with the project build:

1. Extract the downloaded project and open the solution file in Visual Studio.
1. Right-click on the solution and press "Enable NuGet package Restore".
1. Build the project.

For more details, see [How to Run the Examples](https://docs.aspose.com/html/net/getting-started/how-to-run-the-examples/).
